Nstproxy 支持多种编程语言和工具的集成：

```python

proxies = {
    'http': 'http://username:password@proxy.nstproxy.com:port',
    'https': 'https://username:password@proxy.nstproxy.com:port'

response = requests.get('https://api.example.com', proxies=proxies)
```

### Node.js 集成

```javascript

    host: 'proxy.nstproxy.com',
    port: 8080,
    auth: {
```
