## Monad 是什么？

Monad 是一条 100% EVM 兼容的高性能 Layer-1 区块链，由前 Jump Trading 团队成员 Keone Hon 和 Eunice Giarta 于 2022 年创立。它的核心卖点是 **Solana 级别的性能 + 以太坊的兼容性**——这是业界首次有人在保持完整 EVM 兼容的前提下做到 10,000+ TPS 和 1 秒最终确认。

如果你觉得以太坊 L2 已经够快了，那 Monad 的目标是直接把 L1 做到极致。它通过**并行执行**（Parallel Execution）和**异步共识**（MonadBFT）两大技术创新，在不牺牲去中心化的前提下实现了性能飞跃。

## 准备工作

### 1. 钱包

推荐 **MetaMask** 或 **Rabby Wallet**。如果你打算多号操作，Rabby 的多地址管理更友好。

### 2. 添加 Monad 测试网

在 MetaMask 中手动添加网络：

| 参数 | 值 |
|------|-----|
| Network Name | Monad Testnet |
| RPC URL | `https://testnet-rpc.monad.xyz` |
| Chain ID | `10143` |
| Currency Symbol | `MON` |
| Block Explorer | `https://testnet.monadexplorer.com` |

💡 备选 RPC（如果官方 RPC 拥堵）：
- `https://monad-testnet.drpc.org`
- `https://rpc-testnet.monad.xyz`（Alchemy 节点）

### 3. 领测试币

打开 [Monad 官方水龙头](https://faucet.monad.xyz/)，连接钱包，完成人机验证即可领取。

- 每次可领 **0.05 ~ 5 MON**（根据钱包活跃度和 Discord 角色决定）
- 每 **12 小时**可领一次
- 建议每天定时领，积少成多

> ⚠️ 如果官方水龙头额度不够，可以在 Monad Discord 的 `#faucet` 频道申请额外额度，或使用第三方水龙头。

## 核心交互指南

### 第一层：基础交互（必做）

这些是所有空投猎人的基本功，完成度直接影响你的空投权重。

#### 1. DEX 交易

Monad 测试网上部署了多个主流 DEX，交互优先级如下：

- **[Uniswap](https://app.uniswap.org)** — 切换网络到 Monad Testnet，用 MON 换 USDC / WETH 等测试代币。至少做 5-10 笔 swap。
- **[Ambient Finance](https://ambient.finance)** — Monad 生态头部 DEX，做 swap 并添加流动性。
- **[Monorail](https://monorail.xyz)** — Monad 原生的永续合约 DEX。

**操作建议：**
- 每次 swap 不同金额，不要机械地来回换
- 部分 MON 换成稳定币后，去提供 LP（流动性）
- 保持一定的交易频率，比如每天 2-3 笔，持续 2 周以上

#### 2. 借贷协议

- **[Aave](https://app.aave.com)** — 切换到 Monad 网络，存入 ETH/WBTC 等资产，再借出少量稳定币
- **[Morpho](https://app.morpho.org)** — 在 Morpho 上做类似操作

**操作建议：**
- 先存入（Supply），再借出（Borrow），保持健康率在安全范围
- 不要一次性还清，让借贷头寸保持至少 3-5 天
- 借还循环 2-3 次

#### 3. NFT 铸造

Monad 官方推出了限量 **25,555 个**的纪念 NFT，这是目前确定性最高的交互。

1. 访问 [Monad NFT Mint 页面](https://www.monad.xyz/mint)（需要留意官方公告，NFT 可能是分批发售）
2. 用测试网 MON 铸造
3. 在二级市场挂单测试

> 📌 NFT 铸造可能需要白名单或特定条件，关注 Monad 官方 Twitter 获取最新铸造窗口信息。

### 第二层：社交与社区（加分项）

Monad 非常重视社区活跃度，Discord 角色系统是空投权重的重要组成部分。

#### Discord 角色获取

1. 加入 [Monad Discord](https://discord.gg/monad)
2. 进入 `#social-credit` 频道
3. 找到 "Comment Mob" 相关帖子，点击 "Get role" 获取角色
4. 活跃发言，提升 XP 等级
5. 使用 MEE6 机器人查看自己的等级：`/rank`

Monad Discord 有多种角色，包括但不限于：
- **Comment Mob** — 基础社区角色
- **Monartist** — 创作者角色（发 meme、做视频）
- **Dev Monk** — 开发者角色

角色越多、XP 越高，空投分配时权重越大。

### 第三层：高级玩法

如果你想和其他人拉开差距，这些操作值得尝试：

#### 1. 部署智能合约

这是最硬核但也最有效的交互方式。在 Monad 上部署一个简单的合约（哪怕是 Hello World），能显著提升你的"开发者"身份权重。

```solidity
// 一个最简单的计数合约
// 使用 Remix IDE 连接 Monad Testnet 部署即可
contract Counter {
    uint256 public count;
    function increment() public { count++; }
}
```

#### 2. 多钱包策略

如果你打算多号操作，务必注意防女巫（Sybil）：

- ✅ 每个钱包使用**独立 IP**（代理或住宅 IP）
- ✅ 交互模式要有差异化（不同金额、不同时间、不同顺序）
- ✅ 钱包之间不要互转资金
- ✅ 每个钱包绑定独立的 Discord / Twitter 账号
- ❌ 不要在同一天注册所有钱包
- ❌ 不要机械重复完全相同的操作序列

#### 3. 生态项目深度交互

除了主流协议，找 3-5 个早期 Monad 生态项目深度交互：

- **LST 协议**：质押 MON 获取流动性质押代币
- **跨链桥**：从 Sepolia 等测试网跨链资产到 Monad
- **GameFi**：Monad 上已经有一些游戏项目，去玩一下
