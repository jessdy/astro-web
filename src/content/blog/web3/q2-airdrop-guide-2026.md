2026 年的空投季还在继续。Hyperliquid 第二季、Monad 等大项目已经陆续发币，但市场上还有几个顶级项目尚未 TGE（代币生成事件），其中 MegaETH 和 Berachain 是最受关注的两个。

这两个项目都有顶级 VC 背书、主网已上线、生态正在建设中，是 Q2 最值得花时间交互的空投标的。本文带你了解项目背景、参与方式和最大化收益的策略。

## MegaETH：以太坊最快的 L2

MegaETH 是一个基于 Optimism OP Stack 构建的以太坊 Layer 2，主打**实时区块链**概念——目标是把区块时间压缩到毫秒级，TPS 推到 10 万以上。

项目在 2024 年底完成了 2000 万美元种子轮，投资方包括 Dragonfly、Figment Capital、Robot Ventures 等顶级机构。2026 年初主网 Alpha 已上线，但代币 $MEGA 尚未发行。

### 为什么值得撸

- **顶级 VC 背书**：Dragonfly 领投，Vitalik Buterin 也是天使投资人
- **叙事清晰**：主打"实时高性能 EVM"，差异化明显
- **主网已上线**：不是单纯的测试网，是真金白银在跑的链
- **生态空投叠加**：MegaETH 上的 dApp 项目也可能单独发币

### 参与方式

**1. 跨链桥接 ETH 到 MegaETH**

通过官方桥或 RabbitHole 将 ETH 从以太坊主网跨到 MegaETH：

```
官方桥：https://bridge.megaeth.com
RabbitHole：https://rabithole.gg（集成了 MegaETH 桥）
```

建议桥接 0.05-0.1 ETH 即可，不要放太多资金。

**2. 使用 MegaETH 生态 dApp**

这是空投权重的核心。MegaETH 的空投机制与生态互动深度绑定，你需要：

- **每周选择 2-3 个 dApp 进行操作**（Swap、LP、借贷等）
- **使用 Booster 机制**：每周可以给选定的 dApp 应用倍数加成
- **保持活跃**：不要只交互一次就跑，持续参与更重要

目前 MegaETH 生态已上线的主要 dApp：

| dApp | 类型 | 操作建议 |
|------|------|---------|
| MegaSwap | DEX | 完成代币 Swap，添加 LP |
| Teko Finance | 借贷 | 存入资产 + 借出 |
| GTE | 永续合约 | 开仓/平仓交易 |
| Bebop | 聚合器 | 聚合交易 |

**3. 参与社区和测试网任务**

- 关注 MegaETH 官方 Discord 和 Twitter
- 参与 Galxe/Zealy 上的任务活动
- Mint 官方 NFT（如果有的话）

**4. 推荐的交互频率**

```
每周交互计划：
├── 周一：选择本周 2-3 个 dApp，应用 Booster
├── 周二~周四：在选定 dApp 上执行交易
├── 周五：检查是否有新任务或 NFT 活动
└── 持续：保持钱包中有一定余额（0.02 ETH 以上）
```

### 空投预期

根据社区估算和 VC 融资规模类比，$MEGA 的空投价值预期较高。以 2000 万种子轮估值推算，TGE 时 FDV 可能在 10-20 亿美元区间。如果空投占总供应量 5-10%，早期活跃用户可能获得可观的分配。

> ⚠️ 注意：以上为社区推测，实际空投金额和规则以官方公告为准。

## Berachain：流动性证明的 EVM L1

Berachain 是一条基于 Cosmos SDK 构建的 EVM 兼容 Layer 1，核心创新在于**流动性证明（Proof of Liquidity, PoL）**共识机制。

项目完成了 1 亿美元 B 轮融资（估值超 15 亿），投资方包括 Brevan Howard、Framework Ventures、Polychain 等顶级基金。

Berachain 已于 2025 年底上线主网，但代币经济学与传统 L1 完全不同。

### Berachain 的三币模型

Berachain 有三种代币，理解它们是撸空投的前提：

| 代币 | 作用 | 如何获取 |
|------|------|---------|
| **BERA** | Gas 代币，类似 ETH | 交易所购买或跨链 |
| **BGT** | 治理代币，不可转移 | 通过提供流动性获得 |
| **HONEY** | 原生稳定币 | 质押抵押品铸造 |

**关键点：BGT 是空投的核心。** BGT 不可交易，只能通过协议行为获得。项目方已经暗示，BGT 持有者将是未来空投的主要对象。

### 如何获得 BGT

有三种主流方式：

**1. 在 BEX 提供流动性**

BEX 是 Berachain 的原生 DEX。将资产（如 BERA-HONEY 对）存入流动性池，即可获得 BGT 排放。

```
https://bex.berachain.com
```

推荐池子：BERA/HONEY、BERA/WBERA、稳定币对

**2. 在 Bend 借贷协议存入资产**

Bend 是 Berachain 上的原生借贷协议：

```
https://bend.berachain.com
```

存入 BERA 或 HONEY 等资产，既赚利息又拿 BGT。

**3. 在 Berps 永续合约交易**

Berps 是链上永续合约平台，交易行为也会产生 BGT 奖励。

### 交互策略

```
Berachain 撸空投 SOP：
1. 从交易所提 BERA 到 Berachain 钱包
2. 将部分 BERA 换成 HONEY（稳定币）
3. BERA-HONEY 组 LP 存入 BEX → 获得 BGT
4. 在 Bend 存入部分资产 → 获得 BGT
5. 每周在 Berps 做 1-2 笔交易
6. 持续积累 BGT，不要卖
```

BGT 虽然不可直接交易，但可以 1:1 委托给验证节点换取贿赂收益（bribe），或者烧掉换成 BERA。**但为了空投，建议持有 BGT 不动。**

### 空投预期

Berachain 的 BGT 机制让空投的确定性比 MegaETH 更高——因为 BGT 本身就是一种链上积分。你的 BGT 余额直接反映了你对链的贡献，很难被"反撸"。

## 风险提示

1. **空投不确定**：任何项目都可能不发币、改规则、或者空投价值远低于预期
2. **Gas 成本**：虽说交互成本不高，但持续几个月的 Gas 费加起来也不少
3. **合约风险**：新链上的 dApp 可能存在未审计的漏洞
4. **资金安全**：不要把主要仓位放在撸空投的钱包里
5. **女巫检测**：不要用同一个 IP 操作多个钱包，会被判定为女巫攻击
